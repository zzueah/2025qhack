- Gangnam, South Korea

## Problem #1 Prompt

_Abstract_:

A quantum computer returns a line of bitstring per single run. On the current NISQ
machine, however, this bitstring is damaged by noise. To overcome this problem, we
have developed a technique called error mitigation. Error mitigation has shown
promising results even with noisy devices, playing a key component in the claim of
quantum utility era. Meanwhile, it has been somewhat challenging to formulate error
mitigation in sample-based quantum algorithms compared to dealing with expectation
values. This challenge aims to explore a sample-based quantum algorithm based on
one of the most successful error mitigation schemes, configuration recovery. This
includes a case study in sample-based quantum diagonalization (SQD), especially
focusing on the configuration recovery technique. Teams should propose a problem
and implement a sample-based quantum algorithm that solves it, possibly adapting
the configuration recovery method for better fit to the algorithm.

Further details in the reference, problem number 1.
